## About this repository

This repository contains my work on mathematical statistics applied to science and biomedical research.

Here, I explore statistical methods not just theoretically, but through practical implementation using **Python**.

### Goals
* Deepen understanding of statistical foundations.
* Apply mathematical concepts to real-world data analysis.
* Bridge the gap between pure mathematics and Python implementation.

### Tech Stack
* **Language:** Python 3.x
* **Libraries:** NumPy, Pandas, Matplotlib, SciPy, Statsmodels

**Disclaimer:** This is a learning repository. The interpretations and code represent my personal understanding and growth process.
